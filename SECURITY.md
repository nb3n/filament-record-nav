Security reporting and supported versions for `filament-record-nav`.

## Supported Versions

- **2.x** - supported, Filament ^4.0 | ^5.0
- **1.x** - not supported, Filament ^3.0

Requires PHP `^8.2` and Laravel `^10.0 | ^11.0 | ^12.0 | ^13.0`. Upgrade to 2.x before reporting - see the Upgrade Guide in the [README](README.md).

---

## Reporting a Vulnerability

Report privately, not as a public issue or pull request.

- **GitHub** - [private vulnerability reporting](https://github.com/nb3n/filament-record-nav/security/advisories/new)
- **Email** - hello@nben.com.np

Include a description of the impact, reproduction steps, the affected version, and your Laravel/Filament versions.

---

## Response Process

- **Acknowledgment** within 72 hours
- **Assessment** within 7 days
- **Fix** released as a patch, prioritized by severity
- **Credit** in the release notes, with your permission

---

## Out of Scope

- Vulnerabilities in Laravel or Filament itself - report these upstream
- Issues that assume an already compromised admin account
- Documentation and media assets
